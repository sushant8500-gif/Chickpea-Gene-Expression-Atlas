# Chickpea Gene Expression Atlas

The Chickpea Gene Expression Atlas is an interactive resource for visualizing gene expression patterns across chickpea tissues and developmental stages.

This repository contains two versions of the atlas:

1. `shiny_app/` — R Shiny source code for the full interactive app.
2. `docs/` — Static GitHub Pages website version using HTML, CSS, JavaScript, CSV, and SVG files.

## Features

- Gene expression atlas viewer
- Tissue-specific eFP-style SVG visualization
- Multiple color palette options
- Stroke/outline control for atlas tissues
- Bar plot visualization
- Heatmap visualization
- Mean and median expression summaries
- GWAS candidate gene search by chromosome, SNP position, and genomic window
- Downloadable figures and tables

Jain M, Bansal J, Rajkumar MS, Garg R. An integrated transcriptome mapping the regulatory network of coding and long non-coding RNAs provides a genomics resource in chickpea. Communications Biology. 2022;5:1106. PMID: 36261617.
